## 如何使用  
+ 安装`PyExecJS`: `pip2 install PyExecJS`
+ 下载后使用Python2运行QLiker.py 
+ ```nohup python2 QLiker.py >qliker.log&```  
+ ```ls```, 直到出现v.png  
+ 下载v.png,使用手机QQ扫描登陆  
+ 如需更改默认刷新频率，请更改代码第19行（默认180秒，刷新太快网页空间会被封，直接显示为403）  
  
## 功能   
+ 自动点赞，每N秒刷新一次。后台挂机运行
